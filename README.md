## 🚀 Como Fazer Deploy

### 1. No GitHub (primeira vez)

```bash
# Se ainda não tem repositório remoto
git add .
git commit -m "Initial commit: Dra. Fernanda Delosi website"
git branch -M main
git remote add origin https://github.com/seu-usuario/rinoplastia-fernando.git
git push -u origin main
```

### 2. No Vercel (recomendado)

1. Acesse https://vercel.com
2. Clique em "New Project"
3. Conecte seu GitHub
4. Selecione este repositório
5. Clique em "Deploy" (configuração automática)
6. Copie a URL do Vercel (ex: `rinoplastia-fernando.vercel.app`)

### 3. No Netlify (alternativa)

1. Acesse https://netlify.com
2. Clique em "New site from Git"
3. Conecte seu GitHub
4. Selecione este repositório
5. Clique em "Deploy site"

---

## 🌐 Configurar Domínio na Hostinger

1. Acesse seu painel Hostinger
2. Vá para **Domínios** → **Gerenciar DNS**
3. Adicione um registro `CNAME`:
   - **Nome:** `www` (ou `@` para raiz)
   - **Valor:** URL do Vercel/Netlify (ex: `cname.vercel-dns.com` ou `vercel.com`)
4. Salve as alterações
5. Aguarde até 48h para propagação DNS
